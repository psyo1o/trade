run_bot.py

  - 사이클: load_state 후 reconcile_positions_for_cycle(이번 cycle_tag filled 매도 ↔ positions)
  - 자동 매도 체결 후 ledger_apply.persist_position_set/remove (EXIT·HALF·Scale-Out·스윙 FULL 등)
  - persist_position_registration → ledger_apply (BUY 120초 멱등)
  - 핵심 함수: run_trading_bot() = 동기화 → 리스크 → 매도 → 매수
  - 앞단 단계화: _prepare_cycle_state(prune 멱등·bal_read.invalidate), _sync_positions_for_cycle,
    load_state·reconcile_positions_for_cycle, _build_market_context

  - get_safe_balance("KR"|"US"|"COIN") → balance_read (레거시 시장 문자열 모드)
    (macro_snap: PCR·고래·환율 모멘텀·market_buy_allowed / market_buy_block_reason; macro_mult=1.0)
  - 매수 전 RS: _sort_buy_targets_by_rs → alpha_sizing. 비중: _position_ratio_with_vol_target
    (min(1/N, alpha_target_vol/ATR%), config alpha_target_vol 기본 0.02)
  - 본문: helper 2차 + 시장별 3차 → execution/market_cycles/ (매도·매수). run_bot._run_*_buy_cycle / _execute_*_market_buy_twap 은 **위임 래퍼** — 구현은 *_buy_cycle.py·order_executor.py.
  - run_trading_bot: prepare → sync → context → **load 재로드** → reconcile → KR RS → run_kr/us/coin_cycle → save_state.
  - Phase5: 비중 판정 circuit_break.py · 루프 연동 execution/phase5_ops.py (run_bot 위임).
    합산 MDD는 account_circuit_use_total. docs/PHASE5_ACCOUNT_CIRCUIT.md
  - circuit_aux_last_coin_krw / phase5_share_anchor / phase5_pending_liquidation_markets
  - last_kis_display_snapshot (국·미 예수·총평 단일 저장소) · _phase5_aux_sync (ledger_only 시 장부+시세 추정 총평)
  - kis_balance_sync_mode on_trade: 평상시 장부+시세, 매매·강제새로고침·입출금 시 KIS 실조회
  - refresh_circuit_aux_from_brokers … adjust_capital·live sync 시 (KR/US → last_kis_display_snapshot, 코인 → circuit_aux_last_coin_krw)
  - 관측성: [KR …], [US …], [COIN …] 태그. 매수 스캔은 rules.calculate_pro_signals 안에서 [V8],
    스윙 보조는 check_swing_entry 실패 시 [스윙] 한 줄(run_bot에서 출력).
