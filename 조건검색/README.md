# 조건검색 (HTS)

한국투자 HTS에 등록하는 조건검색식 원본입니다. `screener.py` 가 `config.json` 의 `kis_hts_id` 로 **등록된 모든 식**을 조회해 `kr_targets.json` 을 만듭니다.

## 최신 V8 (운영 기준)

| 파일 | 용도 |
|------|------|
| **`v8조건검색 26.05(외국인수급제외 간결화).txt`** | 조건 항목·조합로직 텍스트 (이 문서와 동일) |
| `v8조건검색 26.05(외국인수급제외 간결화).xml` / `.tdf` | HTS 가져오기 |

**조합:** `A and B and ((C and D and E and F) or (G and H))`

- **A·B** — 주가 3,000원 이상, 5봉 평균 거래대금 (공통 유동성)
- **C~F** — 20일선 위·상승, 전일동시간 거래량 200~900%, MACD > Signal → 봇 **V8** (`calculate_pro_signals`)
- **G~H** — 볼밴 하단 상향 돌파, RSI Signal 상향 돌파 → 봇 **스윙** (`check_swing_entry`, V8 실패 후)

**26.05 간결화 변경:** 구 `v8조건검색 26.05.txt` 에 있던 **외국인·기관 5일 순매수**, 시가총액, 순이익 증가율 등 **수급·펀더 블록 제거**.

상세·봇 연동: 저장소 루트 **`README.md` §8 「국장 HTS 조건검색 (V8 26.05)」**.

## 기타 (참고·레거시)

- `v8조건검색 26.05.txt` — 간결화 이전 (3갈래 OR, 외국인/기관 포함)
- `스윙조건검색.txt`, `바닥탈출턴어라운드.txt` — 별도 HTS 식 참고
- `v8조건검색식 외국기관 수급 2번째or에도 추가 26.05.txt` — 실험안 (운영 기본 아님)
